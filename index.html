IGP/VAST Earthquake Report Viewer

Cách dùng: đặt file này cùng cấp với thư mục data/, double-click mở bằng browser, chọn data/articles.jsonl. Nếu muốn xem ảnh local, giữ nguyên cấu trúc data/images/<item_id>/....
